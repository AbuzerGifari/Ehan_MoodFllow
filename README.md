**Ehan MoodFlow** is a next-gen, mobile-first mental health tracker designed with a sleek **Bento-style UI**. It leverages local AI logic to analyze your thoughts, provides voice journaling, and a Zen focus mode—all while staying 100% free and open-source.

---

## ✨ Key Features

- 🧠 **AI Sentiment Analysis:** Automatically detects your mood patterns from your journal entries.
- 🎤 **Voice Journaling:** Speak your mind using the built-in Web Speech API.
- 🧘 **Zen Focus Timer:** Built-in breathing exercise to reduce instant stress.
- 📊 **Insightful Analytics:** Visual mood trends using Chart.js.
- 📱 **Mobile Optimized:** Fluid Bento-grid layout that feels like a native app.
- 🌑 **Glassmorphism Design:** Ultra-modern dark theme with premium gradients.
